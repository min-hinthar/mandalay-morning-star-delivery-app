# Verification Gates (Claude must ask)

Gate A — Choose a direction
- Present at least 3 distinct directions (name + vibe + pros/cons)
- Ask user to choose 1 or hybrid.

Gate B — Palette approval
- Show light/dark palette preview (role table)
- Ask: “Which accent(s) feel most on-brand?”

Gate C — Motion intensity approval
- Provide 3 micro + 1 page transition + 1 scroll sequence
- Ask: “More subtle, or more playful?”

Gate D — 3D intensity approval
- Offer: decorative only vs limited interactive
- Ask which screens get 3D

Gate E — Overlay clickability proof
- Ask user to confirm manual checks passed:
  - header/nav clickable
  - tooltip triggers work
  - cart backdrop blocks clicks
  - ESC closes dialogs, focus trapped

Gate F — Performance sanity
- Ask for confirmation of “no jank on mid device”
