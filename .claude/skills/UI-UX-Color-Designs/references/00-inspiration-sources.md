# Inspiration Sources (public + citeable)

## A) Top brand / product inspiration (visual reference — DO NOT copy assets)
- Nike (marketing + product storytelling patterns)
- Uber / Uber Eats (product UX density, clarity, map + status UI)
- DoorDash (ordering funnel clarity, cart + checkout patterns)
- Panda Express / Chipotle (menu browsing + upsell patterns)
- Airbnb (card systems + photography + trust UI)
- Duolingo (playfulness + character motion + gamified feedback)

When using these, output an “Inspiration Matrix”:
- Brand → Trait → Pattern we adapt → Why it fits our product → What we avoid copying.
