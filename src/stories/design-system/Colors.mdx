# Color Tokens

The Mandalay Morning Star design system uses semantic color tokens for consistent theming across light and dark modes. All colors are defined as CSS custom properties in `tokens.css` and mapped to Tailwind utilities.

## Usage

```css
/* CSS custom property */
.my-element {
  background-color: var(--color-primary);
  color: var(--color-text-inverse);
}
```

```tsx
/* Tailwind utility */
<button className="bg-primary text-text-inverse hover:bg-primary-hover">
  Click me
</button>
```

Toggle the theme using the toolbar above to see how colors adapt between light and dark modes.
