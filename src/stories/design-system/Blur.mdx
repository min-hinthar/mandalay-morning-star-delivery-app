# Blur Tokens

The blur system provides consistent backdrop blur values for glassmorphism effects, modals, and overlay UIs. Blur values are the same in both light and dark modes.

## Blur Scale

From no blur to maximum glass effect.

## Token Reference

| Token | CSS Value | Tailwind Class | Use Case |
|-------|-----------|----------------|----------|
| `--blur-none` | `0px` | `backdrop-blur-none` | No blur |
| `--blur-sm` | `4px` | `backdrop-blur-sm` | Modal backdrop |
| `--blur-md` | `8px` | `backdrop-blur-md` | Light glass |
| `--blur-lg` | `12px` | `backdrop-blur-lg` | Standard glass |
| `--blur-xl` | `20px` | `backdrop-blur-xl` | Heavy glass |
| `--blur-2xl` | `30px` | `backdrop-blur-2xl` | Header glass |
| `--blur-3xl` | `36px` | `backdrop-blur-3xl` | Maximum blur |

## Usage

```css
/* CSS custom property */
.glass-panel {
  background-color: rgba(255, 255, 255, 0.8);
  backdrop-filter: blur(var(--blur-xl));
}
```

```tsx
/* Tailwind utility */
<div className="bg-surface-primary/80 backdrop-blur-xl">
  Glass content
</div>
```

**Note:** Blur tokens have the same values in light and dark modes since blur intensity is not theme-dependent.
