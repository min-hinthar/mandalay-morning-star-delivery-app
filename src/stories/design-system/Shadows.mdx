# Shadow Tokens

The shadow system provides depth and elevation to UI elements. All shadows are theme-aware, using primary brand colors in light mode and pure blacks in dark mode for OLED optimization.

## Token Reference

| Token | Tailwind Class | Use Case |
|-------|----------------|----------|
| `--shadow-xs` | `shadow-xs` | Small elements, badges |
| `--shadow-sm` | `shadow-sm` | Light card elevation |
| `--shadow-md` | `shadow-md` | Default cards |
| `--shadow-lg` | `shadow-lg` | Elevated components |
| `--shadow-xl` | `shadow-xl` | Modals, popovers |
| `--shadow-2xl` | `shadow-2xl` | Maximum elevation |
| `--shadow-inner-sm` | `shadow-inner-sm` | Subtle pressed state |
| `--shadow-inner-md` | `shadow-inner-md` | Active buttons |
| `--shadow-glow-primary` | `shadow-glow-primary` | Primary emphasis |
| `--shadow-glow-success` | `shadow-glow-success` | Success states |
| `--shadow-glow-amber` | `shadow-glow-amber` | Progress bars |
| `--shadow-focus` | `shadow-focus` | Keyboard focus |
| `--shadow-focus-success` | `shadow-focus-success` | Valid input focus |
| `--shadow-focus-error` | `shadow-focus-error` | Invalid input focus |
| `--shadow-nav-top` | `shadow-nav-top` | Cart bar (upward shadow) |
| `--shadow-card` | `shadow-card` | Card components |
| `--shadow-hint-sm` | `shadow-hint-sm` | Small dropdowns |
| `--shadow-hint-md` | `shadow-hint-md` | Larger dropdowns |
| `--shadow-primary` | `shadow-primary` | Primary buttons |
| `--shadow-success` | `shadow-success` | Success buttons |
| `--shadow-warning` | `shadow-warning` | Warning buttons |
| `--shadow-error` | `shadow-error` | Error buttons |

## Usage

```css
/* CSS custom property */
.card {
  box-shadow: var(--shadow-card);
}

.card:hover {
  box-shadow: var(--shadow-card-hover);
}
```

```tsx
/* Tailwind utility */
<div className="shadow-card hover:shadow-card-hover">
  Card content
</div>
```

Toggle the theme using the toolbar above to see how shadows adapt between light and dark modes.
