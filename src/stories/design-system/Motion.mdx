# Motion Tokens

The motion system provides consistent timing and easing for all animations. Use CSS tokens for simple transitions and Framer Motion presets for complex animations.

## Framer Motion Presets

For complex animations, use presets from `@/lib/design-system/tokens/motion.ts`.

| Preset | Type | Config | Use Case |
|--------|------|--------|----------|
| `overlayMotion.modalOpen` | Spring | damping: 25, stiffness: 300 | Modal entrance |
| `overlayMotion.modalClose` | Tween | duration: 0.2, ease: easeIn | Modal exit |
| `overlayMotion.sheetOpen` | Spring | damping: 30, stiffness: 300 | Bottom sheet |
| `overlayMotion.drawerOpen` | Spring | damping: 25, stiffness: 300 | Drawer entrance |
| `overlayMotion.backdrop` | Tween | duration: 0.2, ease: easeOut | Backdrop fade |
| `overlayMotion.dropdown` | Spring | damping: 20, stiffness: 400 | Dropdown menu |
| `overlayMotion.tooltip` | Tween | duration: 0.15, ease: easeOut | Tooltip appear |
| `overlayMotion.toast` | Spring | damping: 20, stiffness: 300 | Toast notification |

**Spring physics note:** Framer Motion spring animations calculate duration dynamically based on damping and stiffness. Do not mix duration tokens with spring configs.

## Token Reference

### CSS Variables

| Token | Value | Use Case |
|-------|-------|----------|
| `--duration-instant` | 0ms | Immediate changes |
| `--duration-fast` | 150ms | Micro-interactions |
| `--duration-normal` | 220ms | Default transitions |
| `--duration-slow` | 350ms | Modals, pages |
| `--duration-slower` | 500ms | Complex animations |
| `--ease-default` | cubic-bezier(0.2, 0.8, 0.2, 1) | Standard UI |
| `--ease-spring` | cubic-bezier(0.34, 1.56, 0.64, 1) | Bouncy/playful |
| `--ease-out` | cubic-bezier(0, 0, 0.2, 1) | Enter animations |
| `--ease-in` | cubic-bezier(0.4, 0, 1, 1) | Exit animations |
| `--ease-in-out` | cubic-bezier(0.4, 0, 0.2, 1) | Symmetric |

## Usage

```css
/* CSS transition */
.button {
  transition: background-color var(--duration-fast) var(--ease-out);
}

.modal {
  transition: opacity var(--duration-slow) var(--ease-in-out);
}
```

```tsx
/* Framer Motion */
import { overlayMotion } from '@/lib/design-system/tokens/motion';

<motion.div
  initial={{ opacity: 0, scale: 0.95 }}
  animate={{ opacity: 1, scale: 1 }}
  transition={overlayMotion.modalOpen}
>
  Modal content
</motion.div>
```

**Reduced motion:** The system respects `prefers-reduced-motion` via `[data-reduce-motion="true"]` which sets all durations to 0.01ms.
