# Typography Tokens

The typography system uses Nunito as the primary font family, providing a rounded, playful feel. Padauk is used for Burmese text.

## Usage

```css
/* CSS custom property */
.heading {
  font: var(--type-h1);
}

.body {
  font-size: var(--text-base);
  line-height: var(--leading-normal);
  font-weight: var(--font-normal);
}
```

```tsx
/* Tailwind utility */
<h1 className="text-3xl font-bold leading-tight">
  Heading
</h1>

<p className="text-base leading-normal">
  Body text
</p>
```
