# Spacing Tokens

The spacing system is based on a 4px grid, providing consistent rhythm throughout the UI. All spacing values are available as CSS custom properties and Tailwind utilities.

## Token Reference

| Token | Value | Pixels | Use Case |
|-------|-------|--------|----------|
| `--space-0` | 0 | 0px | No spacing |
| `--space-px` | 1px | 1px | Borders, hairlines |
| `--space-0-5` | 0.125rem | 2px | Micro spacing |
| `--space-1` | 0.25rem | 4px | Tight spacing |
| `--space-2` | 0.5rem | 8px | Small gaps |
| `--space-3` | 0.75rem | 12px | Medium-small |
| `--space-4` | 1rem | 16px | Standard spacing |
| `--space-6` | 1.5rem | 24px | Section gaps |
| `--space-8` | 2rem | 32px | Large spacing |
| `--space-12` | 3rem | 48px | Component separation |
| `--space-16` | 4rem | 64px | Section separation |
| `--space-24` | 6rem | 96px | Page sections |
| `--space-32` | 8rem | 128px | Major sections |

## Usage

```css
/* CSS custom property */
.card {
  padding: var(--space-4);
  margin-bottom: var(--space-6);
}

.section {
  padding-top: var(--space-16);
  padding-bottom: var(--space-16);
}
```

```tsx
/* Tailwind utility */
<div className="p-4 mb-6">
  Card content
</div>

<section className="py-16">
  Section content
</section>
```
