<task type="auto">
  <name>Task 1: Create driver invite send endpoint</name>
  <files>src/app/api/admin/drivers/invite/route.ts</files>
  <action>
    Create POST endpoint at `/api/admin/drivers/invite`:

    **Request validation:**
    - Body: `{ email: string }` - validate email format with Zod
    - Verify admin authentication via supabase auth

    **Business logic:**
    1. Check if email already has pending invite (not expired, not revoked, not accepted)
       - If exists, return 409 Conflict with message to resend
    2. Check if email already registered as driver
       - If exists, return 409 Conflict
    3. Generate secure token: `crypto.randomBytes(32).toString('hex')`
    4. Calculate expires_at: `new Date(Date.now() + 24 * 60 * 60 * 1000)`
    5. Insert into driver_invites table
    6. Call sendDriverInvite(email, token, expiresAt)
    7. Return 201 with invite details (id, email, expiresAt)

    **Error handling:**
    - 400: Invalid email format
    - 401: Not authenticated
    - 403: Not admin
    - 409: Email already has pending invite or is existing driver
    - 500: Email send failure (log error, still return success if DB insert worked)

  </action>
  <verify>TypeScript compilation succeeds, endpoint structure matches existing API patterns</verify>
  <done>
    - POST /api/admin/drivers/invite creates invite and sends email
    - Validates email, checks duplicates, generates 24-hour token
  </done>
</task>
