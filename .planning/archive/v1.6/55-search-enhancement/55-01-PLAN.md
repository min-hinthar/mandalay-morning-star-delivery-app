    1. Install fuse.js: `pnpm add fuse.js`

    2. Create `src/lib/search/search-config.ts`:
       - Export `FUSE_CONFIG` with Fuse.IFuseOptions<EnrichedMenuItem>:
         - keys: nameEn (weight 3), nameMy (weight 2), descriptionEn (weight 1), _categoryName (weight 0.5), tags (weight 0.5), allergens (weight 0.5)
         - threshold: 0.4 (lenient for short Burmese names per research)
         - distance: 200, ignoreLocation: true
         - includeMatches: true, includeScore: true
         - minMatchCharLength: 2, shouldSort: true
       - Export `CATEGORY_EMOJI_MAP: Record<string, string>` mapping category slugs to emoji fallbacks:
         - 'all-day-breakfast': cooking egg, 'rice-noodles-soups': steaming bowl, 'sides': takeout box, 'curries-a-la-carte': curry rice, 'vegetables': broccoli, 'seafood-curries': shrimp, 'appetizers-salads': green salad, 'drinks': bubble tea
       - Export `SCORE_THRESHOLD = 0.7` (Fuse scores 0=perfect, 1=none; filter results above this)

    3. Create `src/lib/search/use-fuzzy-search.ts`:
       - 'use client' directive
       - Export `EnrichedMenuItem` interface extending `MenuItem` with `_categoryName: string` and `_categorySlug: string`
       - Export `FuseSearchResult` interface: `{ item: EnrichedMenuItem; score?: number; matches?: readonly Fuse.FuseResultMatch[] }`
       - Export `useFuzzySearch(categories: MenuCategory[])` hook:
         - `enrichedItems = useMemo(() => categories.flatMap(cat => cat.items.map(item => ({ ...item, _categoryName: cat.name, _categorySlug: cat.slug }))), [categories])`
         - `fuse = useMemo(() => new Fuse(enrichedItems, FUSE_CONFIG), [enrichedItems])` -- memoized index
         - Export `search(query: string): FuseSearchResult[]` function that calls `fuse.search(query)` and filters by SCORE_THRESHOLD
         - Return `{ search, enrichedItems, fuse }`

    4. Create `src/lib/search/category-helpers.ts`:
       - Export `groupResultsByCategory(results: FuseSearchResult[]): Map<string, FuseSearchResult[]>` -- groups by `_categorySlug`
       - Export `getCategoryEmoji(categorySlug: string): string` -- returns emoji from CATEGORY_EMOJI_MAP or default steaming bowl
       - Export `deriveCategoryTabs(results: FuseSearchResult[], categories: MenuCategory[]): { slug: string; name: string; count: number }[]` -- returns tabs in menu sort order, only categories with matches, includes result count

    5. Create `src/lib/search/index.ts` barrel:
       - Re-export everything from search-config, use-fuzzy-search, category-helpers
