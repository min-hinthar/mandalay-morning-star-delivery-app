For EACH of the 275 files, analyze and categorize:

**Categorization criteria:**

**KEEP** - Must remain client component:
- Uses React hooks (useState, useEffect, useCallback, useMemo, useRef, useContext)
- Uses event handlers (onClick, onChange, onSubmit, etc.)
- Uses framer-motion for animations
- Uses browser APIs (window, document, localStorage)
- Uses client-only libraries (zustand stores, react-query hooks)

**CONVERT** - Can become server component:
- No hooks or event handlers
- Only renders static content
- Only uses server-safe imports
- Wrapper/composition components with no interactivity

**SPLIT** - Needs refactoring:
- Single small hook that could be extracted
- One onClick handler on otherwise static component
- Mixed static content + small interactive element

**LEAF** - Already correctly structured:
- Small, focused interactive component (<100 lines)
- Single responsibility (button, input, toggle)
- Receives props, renders interactive element
