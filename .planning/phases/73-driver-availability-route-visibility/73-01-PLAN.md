1. **Migration** (`supabase/migrations/026_driver_availability.sql`):
   ```sql
   ALTER TABLE drivers
   ADD COLUMN IF NOT EXISTS availability_json JSONB
   DEFAULT '{"available_days": [], "blocked_dates": []}'::jsonb;

   COMMENT ON COLUMN drivers.availability_json IS
   'Driver weekly availability: available_days (recurring day-of-week) and blocked_dates (one-off YYYY-MM-DD)';
   ```

2. **Types** (`src/types/driver.ts`): Add at end of DRIVERS section:
   ```typescript
   export type DayOfWeek = "monday" | "tuesday" | "wednesday" | "thursday" | "friday" | "saturday" | "sunday";

   export interface DriverAvailability {
     available_days: DayOfWeek[];
     blocked_dates: string[]; // ISO date strings "YYYY-MM-DD"
   }
   ```
   Also add `availability_json: DriverAvailability | null;` to `DriversRow`, and optional `availability_json?: DriverAvailability | null;` to `DriversInsert` and `DriversUpdate`.

3. **Shared helper** (`src/lib/availability.ts`): Create with:
   - `DAYS_OF_WEEK` constant: `["monday", "tuesday", "wednesday", "thursday", "friday", "saturday", "sunday"] as const`
   - `DAY_LABELS` map: `{ monday: "Mon", tuesday: "Tue", ... sunday: "Sun" }`
   - `isDriverAvailable(availability: DriverAvailability | null, date: string): boolean` function:
     - If `availability` is null or `available_days` is empty array, return `true` (available all days by default)
     - Check `blocked_dates.includes(date)` -> return `false`
     - Get day-of-week from `new Date(date + "T12:00:00Z").getUTCDay()` mapped to DayOfWeek string
     - Return `available_days.includes(dayOfWeek)`
   - Re-export `DayOfWeek` and `DriverAvailability` from `@/types/driver`

4. **API endpoint** (`src/app/api/driver/availability/route.ts`):
   - **GET**: Use `requireDriver()` auth pattern, `checkRateLimit` with `driverActionLimiter`. Select `availability_json` from `drivers` where `id = driverId`. Return `{ availability: driver.availability_json ?? { available_days: [], blocked_dates: [] } }`.
   - **PATCH**: Same auth/rate-limit. Parse request body. Validate body shape: `available_days` must be array of valid DayOfWeek strings, `blocked_dates` must be array of YYYY-MM-DD strings. Update `drivers` set `availability_json = body` where `id = driverId`. Return `{ availability: body }`.
   - Import patterns from `src/app/api/driver/routes/history/route.ts` (requireDriver, checkRateLimit, driverActionLimiter, logger).
