**1. Create StarRating.tsx:**
- Props: `value: number`, `onChange: (rating: number) => void`, `disabled?: boolean`, `size?: 'sm' | 'md' | 'lg'`
- Renders 5 star icons (lucide `Star`) in a row
- Stars up to `value` are filled (saffron-500 fill), rest are outline (charcoal-300)
- Each star: `m.button` with `whileTap={{ scale: 0.8 }}` and `whileHover={{ scale: 1.2 }}`
- Haptic feedback: `navigator.vibrate?.(30)` on tap
- Accessible: `role="radiogroup"`, each star `role="radio"` with `aria-checked` and `aria-label`
- `'use client'`

**2. Create useDriverRating hook (`src/lib/hooks/useDriverRating.ts`):**
- `useDriverRating(orderId: string, initialRating?: number)` returns `{ rating, submitRating, isSubmitting, error }`
- `initialRating` pre-populates the local `rating` state from API response (for post-delivery read-only revisit)
- `submitRating(rating: number)`: POST to `/api/orders/${orderId}/rating` with `{ rating }`
- On success: update local `rating` state, show toast "Thanks for your rating!"
- On error: show error toast, keep previous state

**3. Create rating API route (`src/app/api/orders/[id]/rating/route.ts`):**
- POST handler:
  - Authenticate user
  - Parse body: `{ rating: number }` -- validate 1-5 integer
  - Verify order exists and belongs to user (or shared access)
  - Verify order status is `delivered`
  - Look up `route_stops` for this order to get `route_id` and then `driver_id` from `routes`
  - Upsert into `driver_ratings`: `{ order_id, driver_id, user_id, rating }`
    - Use upsert (on conflict order_id) so re-rating replaces previous
  - Return 200 with `{ success: true, rating }`

**4. Create DeliveredScreen.tsx:**

- Full-screen overlay component that appears when `orderStatus === "delivered"`
- Auto-transition: use `AnimatePresence` with entry animation
- Celebration animation:
  - Animated checkmark: SVG circle + check path with stroke-dashoffset animation (draw-in effect)
  - Confetti: use CSS keyframe animations with ~20 colored dots falling from top. Keep lightweight (no heavy library). Small colored circles with `position: absolute`, randomized `left`, `animation-delay`, and `animation-duration`.
  - "Delivered!" heading with scale-in animation
- **Wire rating submission at component top:**
  ```typescript
  const { rating, submitRating, isSubmitting } = useDriverRating(orderId, initialRating ?? undefined);
  ```
  Then pass to StarRating component:
  ```tsx
  <StarRating value={rating} onChange={submitRating} disabled={isSubmitting} />
  ```
- Star rating section below: "How was your delivery?" + StarRating component
  - Pre-populate from `initialData.rating` if exists (post-delivery revisit)
- Delivery photo (if exists): show below rating
- "View Order Details" link back to order page
- Post-delivery read-only mode: if visiting from order history, skip celebration animation (detect via query param `?review=true` or simply don't animate if rating already exists)
- `'use client'`
