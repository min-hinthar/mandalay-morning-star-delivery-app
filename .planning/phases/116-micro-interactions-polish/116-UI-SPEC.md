## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Toast internal padding, chip gaps |
| md | 16px | Default element spacing, toast padding |
| lg | 24px | Sticky container padding |
| xl | 32px | Section padding |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Scroll fade gradient overlay: 32px width (w-8) -- narrower than standard spacing, serves as visual indicator
- Touch target minimum: 44px (h-11) -- Phase 113 enforcement, applies to undo button and sticky reorder button
- Safe area inset: `env(safe-area-inset-bottom, 0px)` -- dynamic, device-dependent

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--color-surface-primary` (#ffffff / dark: system) | Page backgrounds, toast surface |
| Secondary (30%) | `--color-surface-secondary` (#fafafa) | Cards, elevated containers, sticky reorder bg |
| Accent (10%) | `--color-primary` (#a41034) | Primary CTA buttons (reorder), active states |
| Destructive | `--color-status-error` (#c45c4a) | Swipe-to-delete background indicator |
| Info | `--color-status-info` (#00979d) | Undo toast type, countdown progress bar |

Accent reserved for:
- Sticky reorder button (primary CTA)
- Toast undo action button text/border
- Countdown progress bar fill

Color usage per feature:
- **Undo toast**: `bg-surface-primary` body, `text-status-info` action button, `bg-status-info` countdown bar
- **Swipe hint bounce**: no color change (position animation only)
- **Scroll fade**: `from-surface-primary to-transparent` gradient (light), `from-surface-elevated to-transparent` (dark)
- **Sticky reorder**: `bg-surface-primary` with `border-t border-border` separator, `shadow-lg` warm tint
- **OG image**: static brand PNG (1200x630), uses brand red #a41034

## Interaction Contracts

### Toast Action Extension (UXPL-01, UXPL-02)

| Property | Value | Source |
|----------|-------|--------|
| Action interface | `action?: { label: string; onClick: () => void }` | D-08 |
| Action button size | min 44px height (h-11) | Phase 113 |
| Action button style | text button, `text-status-info`, `font-semibold` | D-09 |
| Action tap feedback | `scale: 0.97` with `spring.snappyButton` (stiffness 500, damping 30) | D-11 |
| Countdown bar | 100% to 0% over toast duration, `bg-status-info`, height 3px | D-10 |
| Countdown style | Solid fill (not gradient) | Claude's discretion |
| Toast entrance | `spring.default` (stiffness 300, damping 22, mass 0.8) | D-09 |
| Auto-dismiss on action | Prevented -- dismiss only after onClick callback | D-12 |
| Haptic on undo | `triggerHaptic("success")` | D-04 |
| Toast z-index | z-80 (existing token) | Phase 113 |
| Reduced motion | Skip spring entrance; use opacity fade 180ms | useAnimationPreference |

### Cart Undo Mechanism (UXPL-01, UXPL-02)

| Property | Value | Source |
|----------|-------|--------|
| Remove snapshot | `get().items.find(i => i.cartItemId === id)` before removeItem | D-02 |
| Clear snapshot | `[...get().items]` before clearCart | D-03 |
| Undo window | 5000ms (toast duration) | D-04, UXPL-01 |
| Restore (single) | `addItem(snapshot)` | D-02 |
| Restore (clear) | `set({ items: snapshot })` | D-03 |
| Debounce | Store-level debounce on undo clicks | D-05 |
| Edge case | Validate item in menu before restore; error toast if unavailable | D-06 |
| Dismiss behavior | Toast dismiss on undo click = immediate restore | D-05 |

### Swipe Hint Animation (UXPL-03)

| Property | Value | Source |
|----------|-------|--------|
| Trigger | First cart item render when `!localStorage.getItem("swipeHintSeen")` | D-13 |
| Animation | `x: [0, -30, 0]` | D-14 |
| Spring | `spring.ultraBouncy` (stiffness 500, damping 12, mass 0.8) | D-14 |
| Delay | 800ms after mount | D-14 |
| Frequency | Once per user (localStorage flag set after completion) | D-15 |
| PanInfo guard | `if (!info?.offset \|\| !info?.velocity) return` | D-16 |
| Touch action | Drag handle: `touch-none`; content: `pan-y` | D-17 |
| Reduced motion | Skip bounce; no visual hint | useAnimationPreference |

### Dietary Scroll Indicators (UXPL-04)

| Property | Value | Source |
|----------|-------|--------|
| Left gradient | `bg-gradient-to-r from-surface-primary to-transparent w-8` | D-19 |
| Right gradient | `bg-gradient-to-l from-surface-primary to-transparent w-8` | D-19 |
| Dark mode | `from-surface-primary dark:from-surface-elevated` | D-21 |
| Show left threshold | `scrollLeft > 10` | D-20 |
| Show right threshold | `scrollLeft < scrollWidth - clientWidth - 10` | D-20 |
| Position | `absolute`, `pointer-events-none`, `z-10` | D-20 |

| Pattern source | CategoryTabs `updateFadeIndicators` callback | D-18 |
| Gradient width | 32px (w-8) | Claude's discretion (8px baseline adjusted up for visibility) |

## Animation Token Reference

All animations in this phase use existing motion tokens. No new tokens required.

| Token | Values | Used By |
|-------|--------|---------|
| `spring.default` | stiffness 300, damping 22, mass 0.8 | Toast entrance, undo restore |
| `spring.snappyButton` | stiffness 500, damping 30, mass 0.8 | Action button tap (scale 0.97) |
| `spring.ultraBouncy` | stiffness 500, damping 12, mass 0.8 | Swipe hint bounce |

| `duration.normal` | 0.18s (180ms) | Reduced motion fallback fade |
| `duration.fast` | 0.12s (120ms) | Scroll fade indicator transition |

## Accessibility Checklist

- [ ] Undo button meets 44px min touch target (h-11)
- [ ] Undo button has `focus-visible:ring-2 ring-primary ring-offset-2`
- [ ] Toast action button has accessible label (`aria-label` if icon-only; text label sufficient)
- [ ] Scroll fade indicators are `pointer-events-none` (do not block chip interaction)
- [ ] Sticky reorder button keyboard-accessible (inherits from existing Button component)
- [ ] All animations respect `prefers-reduced-motion` via `useAnimationPreference()`
- [ ] Swipe hint bounce skipped under reduced motion
- [ ] OG image has meaningful alt text in meta tags
- [ ] Countdown bar has `aria-hidden="true"` (decorative, time communicated via toast duration)
