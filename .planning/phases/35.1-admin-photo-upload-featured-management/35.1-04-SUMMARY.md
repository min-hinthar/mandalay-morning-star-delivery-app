## Decisions Made

- **Static icon map for DynamicIcon:** The `icons` object from lucide-react caused Turbopack build errors. Created static ICON_MAP with 30 common icons supporting both kebab-case and PascalCase names.
- **Empty state handling:** Shows "Menu Coming Soon" when no featured sections have items, still displays "See Full Menu" CTA.
- **Removed full menu grid from homepage:** CategoryTabs and full menu browsing moved to /menu page only.
