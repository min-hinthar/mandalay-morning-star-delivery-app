## Decisions Made
- **Simplified draft workflow:** Instead of separate draft/live tables, uses a boolean flag per section. Any PATCH sets `has_unpublished_changes = true`, publish clears all flags.
- **Iframe preview:** Shows actual homepage in iframe with device simulation. No special preview mode needed since changes are saved immediately (just not "published").
- **Google Drive URL verification:** Server-side HEAD request verifies image accessibility before saving.
