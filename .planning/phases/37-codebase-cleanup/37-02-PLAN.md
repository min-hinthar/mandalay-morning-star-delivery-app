    Fix: Create `src/components/ui/admin/settings/settings-types.ts` with shared types.

    Root cause: ui/index.ts > cart/index.ts > CartDrawer.tsx > ClearCartConfirmation.tsx > imports from ui/
    Fix: Change imports in ClearCartConfirmation.tsx and CartDrawer.tsx to use direct file imports instead of barrel imports from @/components/ui.
    - Import specific components directly: `import { Button } from "@/components/ui/Button"`
    - NOT from barrel: `import { Button } from "@/components/ui"`

    **Cycle 6: Checkout barrel cycle**
    Root cause: checkout/index.ts > AddressStepV8.tsx > ui/index.ts
    Fix: Change AddressStepV8.tsx to use direct imports instead of ui barrel.

    **Cycle 7-8: Menu barrel cycle**
    Root cause: menu/index.ts > FeaturedCarousel > CardImage/UnifiedMenuItemCard > imports from ui/
    Fix: Change CardImage.tsx and UnifiedMenuItemCard.tsx to use direct imports.

    **Cycle 9: Navigation barrel cycle**
    Root cause: ui/index.ts > navigation/index.ts > AppShell.tsx > MobileMenu.tsx > imports from ui/
    Fix: Change MobileMenu.tsx and AppShell.tsx to use direct imports.

    **Pattern for all fixes:**
    ```typescript
    // BEFORE (creates cycle via barrel)
    import { Button, Modal, Input } from "@/components/ui";

    // AFTER (direct imports, no cycle)

    import { Modal } from "@/components/ui/Modal";
    import { Input } from "@/components/ui/Input";
    ```
