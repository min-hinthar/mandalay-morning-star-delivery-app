Define the following TypeScript types (snake_case fields for JSON consistency with Supabase/Stripe):

```typescript
type ServiceName = "supabase" | "stripe" | "resend";
type StatusLevel = "healthy" | "degraded" | "down";

interface ServiceStatus {
  status: StatusLevel;
  configured: boolean; // Env vars present
  connected?: boolean; // Deep check result (only with ?deep=true)
  latency_ms?: number; // Deep check latency
  error?: string; // Error message (redacted of secrets)
}

interface RouteStatus {
  path: string;
  reachable: boolean;
  status_code?: number; // Only with ?deep=true
}

interface EnvCheckResult {
  configured: number; // Count of configured vars
  missing: string[]; // Names of missing vars (never values)
  total: number; // Total required vars
  all_critical_present: boolean; // All critical-tier vars present
}

interface HealthResponse {
  status: StatusLevel;
  production_ready: boolean;
  timestamp: string;
  version: string;
  environment: string;

  services: {
    supabase: ServiceStatus;
    stripe: ServiceStatus;
    resend: ServiceStatus;
  };

  routes: {
    auth_callback: RouteStatus;
    stripe_webhook: RouteStatus;
  };

  env: EnvCheckResult;
}
```

Export all types.

**Secret redaction helper** (module-private):

```typescript
function redactSecrets(message: string): string {
  return message
    .replace(/(sk_(?:live|test)_)[a-zA-Z0-9]+/g, "$1***")
    .replace(/(pk_(?:live|test)_)[a-zA-Z0-9]+/g, "$1***")
    .replace(/(re_)[a-zA-Z0-9]+/g, "$1***")
    .replace(/(whsec_)[a-zA-Z0-9]+/g, "$1***")
    .replace(/eyJ[a-zA-Z0-9_-]+\.eyJ[a-zA-Z0-9_-]+\.[a-zA-Z0-9_-]+/g, "eyJ***")
    .replace(/(https?:\/\/)[^:]+:[^@]+@/g, "$1***:***@");
}
```

**`checkSupabase()`:**

- Import `createPublicClient` from `@/lib/supabase/server`
- Wrap in try/catch. On success: query `supabase.from("app_settings").select("key").limit(1)`
- If query returns error: status `"degraded"`, connected false, include error.message
- If exception thrown: status `"down"`, connected false, include redacted error
- Return `ServiceStatus` with `latency_ms` (Date.now() delta)
- IMPORTANT: Wrap `createPublicClient()` itself in try/catch to handle missing env vars gracefully

**`checkStripe()`:**

- Import `stripe` from `@/lib/stripe/server`
- Call `stripe.balance.retrieve()`. Stripe proxy throws if STRIPE_SECRET_KEY is missing.
- Success: `"healthy"`, connected true
- Exception: `"down"`, connected false, redacted error
- Return `ServiceStatus` with `latency_ms`

**`checkResend()`:**

- Import `getResendClient` from `@/lib/email/client`
- Call `resend.domains.list()`. Throws if RESEND_API_KEY missing.
- If `domains.list()` returns error property: `"degraded"`, connected false
- If exception: `"down"`, connected false, redacted error
- Return `ServiceStatus` with `latency_ms`

**`checkRoutes(origin: string)`:**

- Takes `origin` string (from `request.nextUrl.origin`)
- Check two routes: `/auth/callback` and `/api/webhooks/stripe`
- Use `fetch(origin + path, { method: "HEAD", signal: AbortSignal.timeout(5000) })`
- IMPORTANT: non-404 = reachable. Auth callback returns 302 (redirect). Webhook returns 405 (POST only). Both are valid "reachable" responses.
- Return `{ auth_callback: RouteStatus, stripe_webhook: RouteStatus }`

**30-second in-memory cache for deep check results:**

- Module-level `let cachedResult: { data: ...; timestamp: number } | null = null`
- `const CACHE_TTL_MS = 30_000`
- Before running deep checks, check if cache is valid. If so, return cached data.
- After deep checks complete, store results in cache.
- Export a `runDeepChecks(origin: string)` function that orchestrates all 4 checks via `Promise.allSettled()`, applies cache, and returns the combined result.
