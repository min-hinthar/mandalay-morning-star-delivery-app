  <action>
    Create a reusable Tabs component following CategoryTabs pattern but more generic:

    ```typescript
    interface Tab {
      id: string;
      label: string;
      icon?: React.ReactNode;
    }

    interface TabsProps {
      tabs: Tab[];
      activeTab: string;
      onTabChange: (tabId: string) => void;
      className?: string;
    }
    ```

    Features:
    - Horizontal layout with animated pill indicator (layoutId)
    - Support for icons in tabs
    - Motion animations
    - Accessible: role="tablist", aria-selected
    - Works on mobile (scrollable if needed)

    Implementation:
    ```tsx
    <div role="tablist" className="flex gap-1 p-1 bg-surface-secondary rounded-card-sm">
      {tabs.map((tab) => (
        <button
          key={tab.id}
          role="tab"
          aria-selected={activeTab === tab.id}
          onClick={() => onTabChange(tab.id)}
          className={cn(
            "relative px-4 py-2 rounded-input font-body text-sm transition-colors",
            activeTab === tab.id ? "text-text-primary" : "text-text-secondary hover:text-text-primary"
          )}
        >
          {activeTab === tab.id && (
            <motion.div
              layoutId="activeTab"
              className="absolute inset-0 bg-surface-primary rounded-input shadow-sm"
              transition={{ type: "spring", bounce: 0.2, duration: 0.4 }}
            />
          )}
          <span className="relative flex items-center gap-2">
            {tab.icon}
            {tab.label}
          </span>
        </button>
      ))}
    </div>
    ```

            onChange={(operations) => setSettings(s => ({ ...s, operations }))}
