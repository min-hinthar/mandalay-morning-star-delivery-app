## Decisions Made

- **Reusable Tabs component**: Created generic Tabs component (not CategoryTabs-specific) for use across admin pages
- **Currency display pattern**: Store values in cents, display as dollars with conversion helpers
- **Change detection**: Deep JSON comparison between original and current settings
- **Unsaved warning**: Both UI banner (fixed bottom) and beforeunload event
- **Toggle switch pattern**: Custom button with role="switch" and aria-checked, reused across forms
- **Tasks 2+3 combined commit**: Forms are tightly coupled to SettingsClient, committed together
