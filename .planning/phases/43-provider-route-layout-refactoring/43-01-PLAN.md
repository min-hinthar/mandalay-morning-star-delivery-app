<objective>
Move CartBar, CartDrawer, and FlyToCart from global providers.tsx into (public) and (customer) route-group layouts. This removes ~60KB of cart component code (plus transitive GSAP/framer-motion sub-trees) from admin, driver, and auth route bundles.

Purpose: Bundle size reduction for non-customer routes. Cart components are only needed on pages where users browse/purchase (home, menu, cart, checkout, account, orders). Admin, driver, and auth routes pay zero cost for cart UI.

Output: CartOverlays.tsx wrapper, two route-group layouts, cleaned providers.tsx, updated CartIndicator with fallback behavior.
</objective>
