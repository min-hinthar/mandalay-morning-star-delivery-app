  <name>Task 1: Add E2E job to CI workflow</name>
  <files>.github/workflows/ci.yml</files>
  <action>
Add E2E test job to ci.yml after the build job:

```yaml
e2e:
  name: E2E Tests
  runs-on: ubuntu-latest
  needs: [build]
  steps:
    - uses: actions/checkout@v4
    - uses: pnpm/action-setup@v4
      with:
        version: 10
    - uses: actions/setup-node@v4
      with:
        node-version: 20
        cache: pnpm
    - run: pnpm install --frozen-lockfile
    - name: Install Playwright browsers
      run: pnpm exec playwright install --with-deps chromium
    - name: Download build artifacts
      uses: actions/download-artifact@v4
      with:
        name: nextjs-build
        path: .next
    - name: Run E2E tests
      run: pnpm test:e2e --project=chromium
      env:
        CI: true
    - name: Upload test results
      if: always()
      uses: actions/upload-artifact@v4
      with:
        name: playwright-report
        path: playwright-report/
        retention-days: 7
```

Key decisions:

- Uses build artifact from build job (avoids rebuilding)
- Only chromium project (mobile Chrome not needed in CI — already tested locally)
- Uploads playwright-report for debugging failures
- `if: always()` ensures report upload even on failure
