**ci.yml changes (Lighthouse + path filter):**

1. Add a `changes` job at the top of jobs using `dorny/paths-filter@v3`:

   ```yaml
   changes:
     name: Detect Changes
     runs-on: ubuntu-latest
     permissions:
       pull-requests: read
     outputs:
       src: ${{ steps.filter.outputs.src }}
     steps:
       - uses: actions/checkout@v4
       - uses: dorny/paths-filter@v3
         id: filter
         with:
           filters: |
             src:
               - 'src/**'
               - 'public/**'
               - 'package.json'
               - 'pnpm-lock.yaml'
               - 'next.config.ts'
               - 'lighthouserc.js'
               - 'postcss.config.*'
   ```

2. Update lighthouse job:
   - Add `changes` to needs: `needs: [build, changes]`
   - Update if condition: `if: github.event_name == 'pull_request' && needs.changes.outputs.src == 'true'`

3. Upgrade ALL jobs from `node-version: 20` to `node-version: 22` (Node 20 EOL April 2026, Node 22 is current LTS and matches Vercel runtime).
