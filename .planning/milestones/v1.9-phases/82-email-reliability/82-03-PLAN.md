  <action>
Create `GET /api/admin/emails/stats` endpoint:

1. Standard admin auth + rate limiting pattern
2. Run three aggregation queries in parallel using `Promise.all()`:

   **Today stats:**
   ```typescript
   const todayStart = new Date();
   todayStart.setHours(0, 0, 0, 0);

   supabase
     .from("notification_logs")
     .select("status", { count: "exact" })
     .gte("created_at", todayStart.toISOString())
   ```
   Then group by status in JS (Supabase doesn't support GROUP BY directly in PostgREST). Alternative: use `.rpc()` with a raw SQL function, but simpler to fetch all today's rows and count in JS since volume is low (~50-200 emails/day max for this business).

   Actually, the simplest approach: run 4 queries per time range (sent, delivered, failed, bounced) with `.eq("status", X).select("id", { count: "exact", head: true })`. This avoids fetching rows entirely — just counts.

   **Efficient approach — 12 count queries in parallel:**
   ```typescript
   const statuses = ["sent", "delivered", "failed", "bounced"] as const;
   const ranges = {
     today: todayStart.toISOString(),
     week: weekStart.toISOString(),
     allTime: null,
   };

   const queries = Object.entries(ranges).flatMap(([range, since]) =>
     statuses.map(async (status) => {
       let q = supabase
         .from("notification_logs")
         .select("id", { count: "exact", head: true })
         .eq("status", status);
       if (since) q = q.gte("created_at", since);
       const { count } = await q;
       return { range, status, count: count ?? 0 };
     })
   );
   const results = await Promise.all(queries);
   ```

3. Shape response:
   ```typescript
   interface EmailStats {
     today: { sent: number; delivered: number; failed: number; bounced: number };
     week: { sent: number; delivered: number; failed: number; bounced: number };
     allTime: { sent: number; delivered: number; failed: number; bounced: number };
   }
   ```

4. Calculate week start as Monday of current week.

Return `NextResponse.json({ stats })`.
