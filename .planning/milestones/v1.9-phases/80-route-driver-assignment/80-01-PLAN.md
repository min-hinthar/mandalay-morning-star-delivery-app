    **clustering.ts** (`src/lib/utils/clustering.ts`):

    1. Import `calculateHaversineDistance` from `@/lib/utils/eta`.
    2. Define `KITCHEN_LOCATION = { lat: 34.0858, lng: -117.8896 }`.
    3. Define `CLUSTER_RADIUS_KM = 2.0` (~1.2 miles).
    4. Define `ROAD_FACTOR = 1.3`, `AVG_SPEED_KMH = 35 * 1.60934`, `STOP_DURATION_MIN = 5`.
    5. Define `CLUSTER_COLORS` array: 6+ distinct colors using design-token-compatible values (e.g., hsl strings or named CSS custom property references).

    6. Interfaces:
       ```typescript
       export interface ClusterableOrder {
         id: string;
         lat: number | null;
         lng: number | null;
       }
       export interface OrderCluster {
         label: string;
         color: string;
         orderIds: string[];
         centroid: { lat: number; lng: number };
       }
       export interface RouteDurationEstimate {
         durationMinutes: number;
         distanceKm: number;
       }
       ```

    7. `clusterOrders(orders: ClusterableOrder[]): OrderCluster[]`:
       - Filter out orders with null lat/lng (they go in a separate "Location Unknown" bucket, NOT returned as a cluster).
       - Sort remaining by latitude for stable grouping.
       - Greedy clustering: for each order, find nearest existing cluster centroid. If distance <= CLUSTER_RADIUS_KM, add to cluster and recalculate centroid. Otherwise, create new cluster.
       - Generate labels using `generateClusterLabel(centroid)` helper.
       - Assign colors from CLUSTER_COLORS by index (modulo wrap).

    8. `generateClusterLabel(centroid: {lat:number,lng:number}, count: number): string`:
       - Compare centroid to KITCHEN_LOCATION.
       - Determine cardinal direction: N/S based on lat diff, E/W based on lng diff, combined (e.g., "Northeast").
       - Return `"${direction} — ${count} order${count !== 1 ? 's' : ''}"`.

    9. `estimateRouteDuration(stops: Array<{lat:number,lng:number}>): RouteDurationEstimate`:
       - If 0 stops: return { durationMinutes: 0, distanceKm: 0 }.
       - If 1 stop: return { durationMinutes: STOP_DURATION_MIN, distanceKm: 0 }.
       - Sum sequential Haversine distances. Multiply by ROAD_FACTOR. Compute driving time from AVG_SPEED_KMH. Add STOP_DURATION_MIN per stop.
       - Return rounded values.

    10. `getUnclusteredOrders(orders: ClusterableOrder[]): string[]`:
        - Return IDs of orders with null lat or lng. Used by UI to show "Location Unknown" group.
