     a. Check if cart has items via `useCartStore.getState().items.length`

     d. Call `POST /api/account/orders/${orderId}/reorder`
     e. On success: `clearCart()`, then `addItem()` for each returned item
     f. Show toast with warning count: "2 of 5 items unavailable -- added 3 to cart" (per user decision)
     g. Open cart drawer via `useCartDrawer.getState().open()`
   - Expose: `{ reorder, confirmReorder, cancelReorder, isLoading, showConfirmation, pendingOrderId }`
   - Import toast from `@/lib/hooks/useToastV8`
   - Use `triggerHaptic` on successful reorder
